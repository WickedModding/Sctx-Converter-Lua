# SCTX-CONVERTER FOR LUA

Sctx-Converter - Simple tool sctx2png, uses [Sctx-Converter](https://github.com/Daniil-SV/SCTX-Converter/tree/main) By [Daniil Sivi](https://github.com/Daniil-SV)

# Operating System Support
Linux(Wine), Windows

# Installing Wine for Linux
Arch
``` sh
sudo pacman -S Wine
```
Ubuntu
``` sh
sudo apt install Wine
```
